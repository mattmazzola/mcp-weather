# Tempate Repository for Python project using UV

## Setup Steps

1. Create repo from template
2. Clone the new repo
3. Rename project.code-workspace to your_project_name.code-workspace
4. Open the workspace in VSCode
5. Open the dev container
6. Run `uv init` to initialize the project
7. Add dependencies
8. Uncommend the `uv sync` in `.devcontainer/postCreateCommand.sh`
9. Rebuild the dev container
10. Rewrite this README.md to reflect your project details!
